Como tenemos la clave AES del script, podemos descifrar todos los mensajes que tenemos en wireshark DNS, con un simple código en python:

```python
from Crypto.Cipher import AES
import base64

key = base64.b64decode("a1E4MUtycWswTmtrMHdqdg==")

def decrypt(bas64):
	enc = base64.b64decode(bas64)
	i, t = enc[:16], enc[16:]
	return AES.new(key, AES.MODE_CBC, i).decrypt(t)
```
